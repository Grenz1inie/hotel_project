> **项目名称**：智能酒店预订与管理系统

> **演示版本**：v1.0  
> **文档更新日期**：2025年10月9日

## 📋 演示准备清单

### 启动前准备

1. **后端服务启动**
   ```bash
   # 进入后端目录
   cd hotel-backend/hotel-backend
   
   # 启动Spring Boot应用
   mvnw.cmd spring-boot:run
   
   # 确认启动成功（看到以下日志）
   ✓ Started HotelBackendApplication in X.XXX seconds
   ✓ Tomcat started on port(s): 8080
   ```

2. **前端服务启动**
   ```bash
   # 进入前端目录
   cd hotel-frontend
   
   # 启动开发服务器
   npm start
   
   # 自动打开浏览器访问 http://localhost:3000
   ```

3. **数据库确认**
   - MySQL 8.0 运行中
   - 数据库已初始化（schema.sql 已执行）
   - 测试数据已就绪

4. **演示账号准备**
   - **普通用户账号**：`user` / `123456`
   - **管理员账号**：`admin` / `123456`

## 🔧 附录：常见问题处理

### Q1: 页面加载慢怎么办？
**A**: 开发环境下首次加载会慢一些，可以说明生产环境会进行代码分割、CDN加速等优化。

### Q2: VR预览不显示怎么办？
**A**: 检查房间是否配置了VR资源URL，如果没有可以说明这是演示数据限制，实际部署时会为所有房型配置VR资源。

### Q3: 空置曲线数据太少怎么办？
**A**: 可以说明演示环境数据有限，实际生产环境会有更丰富的历史数据，曲线会更平滑准确。

### Q4: 退款后VIP没降级怎么办？
**A**: 切换到其他标签页再切回来，或者点击"检查VIP升级"按钮手动触发同步。

### Q5: 管理员看不到订单怎么办？
**A**: 确认是否使用了正确的管理员账号，检查订单是否已创建且数据库有记录。
